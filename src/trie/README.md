# Trie
- Also known as prefix tree or digital tree
- Its a data structure used to associative array where keys are usually string.

## Versions
- Standard trie
- Compressed trie
- Suffix Tree

## Operations
- set(str, value)   : O(m)
- get(str)          : O(m)
- delete(str)       : O(m)
 
m - len(str)
